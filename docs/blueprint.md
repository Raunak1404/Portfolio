# **App Name**: Animated Ascent

## Core Features:

- Animated Home: Home page with animated intro, scroll-triggered effects powered by Anime.js.
- Interactive Timeline: Interactive timeline showcasing life events (school, jobs, achievements) with Anime.js animations.
- Animated Project Showcase: Projects page featuring a grid layout. Project cards animate on hover.
- Animated Poetry Display: Display poems with Anime.js line-by-line animations.
- Animated contact form: A contact form that smoothly shows the status after submitting the form

## Style Guidelines:

- Primary color: Muted blue (#64B5F6) to give a calm, reliable feel.
- Background color: Very light blue (#F0F8FF). Almost white to ensure great readability
- Accent color: Light green (#A5D6A7) to give the portfolio a touch of nature and harmony.
- Clean and modern sans-serif fonts for readability and a professional look.
- Use consistent line-style icons for a clean and modern aesthetic.
- Spacious and clean layout with a focus on whitespace for a modern feel.
- Smooth and subtle animations powered by Anime.js for interactive elements and page transitions.
